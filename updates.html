Updates & Changelog

See what's new in the Smart Exam Schedule tool.

March 2026

Version 3.0 Latest

  • AI Upgraded: Swapped out manual API keys for seamless, built-in Puter.js AI logic.
  • New Architecture: Separated the landing page and the main tool (`sp.html`) for faster loading.
  • Modern UI: Added mesh gradients, glassmorphism cards, and smoother animations.
  • Changelog Page: Added this beautiful timeline view to track features!
February 2026

Version 2.0

  • Color Themes: Added ability to choose card themes (Blue, Emerald, Purple, etc.)
  • PDF Export: Integrated jsPDF to generate selectable-text A4 and Desktop PDF schedules.
  • Image Export: Added html2canvas for pixel-perfect PNG schedule generation.
  • Settings Modal: Added a configuration tab to change University name and Term details.
January 2026

Version 1.0

  • Initial Release: Core schedule tracker built with pure HTML, CSS, and JS.
  • Local Storage: Exam data saves securely on the device without needing a database.
  • Responsive UI: Mobile-first approach ensuring it works on all screen sizes.
